# Customer Order Analysis Project

## Overview
This project analyzes customer order data to understand customer behavior, revenue trends, and order patterns.

## Tools Used
- Python
- SQL (via Python)
- Power BI
- Excel

## Project Structure
- python_sql/ → data processing and SQL analysis
- data/ → dataset files
- powerbi/ → dashboard
- customer_orders.db → database file

## Key Insights
- Identified top customers contributing to revenue
- Analyzed repeat vs one-time customers
- Found sales trends over time

## SQL Usage
SQL queries were executed within Python scripts.
